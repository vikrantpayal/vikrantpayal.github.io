 Kubernetes Certifications / CKA prep notes

Notes from CKA prep..
kubectl command is important. Learn to use it. There is no access to the web dashboard UI. The exam is completely hands-on, no multiple choice questions. Other important things to learn, besides kubectl are etcdctl and kubeadm.

YAML syntax knowledge is essential. Basic commands of vi and syntax of GNU Bash will be needed during the test.



Tips for the test:

  1. The bash autocomplete and kubectl alias are already configured for the test.
  2. You are not required to create users (RBAC) during the test. Users are pre-configured
