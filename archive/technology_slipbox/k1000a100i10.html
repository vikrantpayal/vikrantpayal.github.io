tech slipbox/kubernetes/config
 Commands to fetch information, list things, find status



To find out the current K8s context:
kubectl config current-context


What is the version of K8s?
kubectl version


How can I list all available commands and their shortnames?
kubectl api-resources


How do I get a detailed description of any resource?
kubectl describe < resource-name >
- - this works on pods, services, events.
- - Here's a good blog on the various ways kubectl describe can be used.
- - kubectl get is the chota bhai of kubectle describe
--For example, if you have a service account and you want to get all the details of it, use
kubectl describe serviceaccount mysrvcaccntnam


How do I filter objects by some search term while using describe or get?
kubectl describe pods | grep -C 10 "author=vikrant payal"
kubectl get pods -o yaml | grep -C 5 labels:


How can I get help on any command?
kubectl create --help

How can I find out which fields are defined for a resource?
kubectl explain < resource-name >
- - this command is better than --help


How can I get a list of all service accounts?
kubectl get serviceaccounts


To get secrets
kubectl get secrets
